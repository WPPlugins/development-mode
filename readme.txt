Uses Sunrise theme on Dashboard and Frontend to visually represent development mode


== Description ==

This utility plugin changes the theme of the current user to Sunrise, and uses the Sunrise color scheme when browsing the front-end, to help ensure you are viewing and editing the Development Site when making crucial changes to a site.

This plugin also adds a menu item to the admin bar to further confirm that you are on the Development Site.


== Installation ==

Installation:

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
